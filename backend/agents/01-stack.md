# Stack Tecnológico

- **Framework:** FastAPI
- **ORM:** SQLAlchemy
- **DB:** SQLite
- **Validación:** Pydantic
- **Auth:** JWT (python-jose + passlib)
- **Servidor:** Uvicorn
