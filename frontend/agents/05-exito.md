# Criterios de Éxito

1. Usuario puede registrarse e iniciar sesión
2. 3 tabs accesibles y funcionan correctamente
3. Formularios envían datos con JWT token
4. Dashboard muestra datos correctamente
5. Métricas (promedio, ahorro) se calculan bien
6. Gráfica muestra ingresos vs gastos
7. App responsive
8. Sin errores en consola
