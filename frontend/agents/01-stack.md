# Stack Tecnológico

- **Framework:** React 18 (Vite recomendado)
- **Estilos:** Tailwind CSS
- **Gráficos:** Recharts (recomendado) o Chart.js
- **Comunicación:** Fetch API o Axios
- **Estado:** React hooks o Context API
