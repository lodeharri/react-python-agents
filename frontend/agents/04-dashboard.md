# Dashboard

## Métricas (KPIs)
- **Promedio Gasto Mensual:** Total gastos / meses con gastos
- **Ahorro Total:** Total Ingresos - Total Gastos

## Componentes Visuales
- **Gráfica:** Barras o línea comparando ingresos vs gastos por mes
- **Tabla:** Fecha, Tipo, Monto, Motivo

---

# UI/UX

- Responsive (móvil + escritorio)
- Navbar visible con 3 tabs
- Pantalla login/registro antes de acceder
- Feedback visual (loading, éxito, error)
- Colores: verde=ingresos, rojo=gastos
- Tablas con scroll si hay muchos registros
